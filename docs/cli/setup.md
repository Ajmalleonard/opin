# `opin setup`

Initialize `~/.opin/opin.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
opin setup
opin setup --workspace ~/.opin/workspace
```

To run the wizard via setup:

```bash
opin setup --wizard
```
